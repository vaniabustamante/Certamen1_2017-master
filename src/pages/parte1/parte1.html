Parte1

1.

Definir una función max_cinco(a, b, c, d, e), que tome cinco números como argumentos y devuelva los tres mayores en orden.

2.

Definir una función palindromo(palabra) que reconoce palabras palíndromos (es decir, palabras que tienen el mismo aspecto escritas invertidas), ejemplo: palindromo("seres") tendría que devolver True.

3.

Desarrollar una función cuadratica(a, b, c) que permita ingresar los valores de a, b y c de una función cuadratica y entregue las soluciones para la ecuación a(x^2) + bx + c = 0

Importante: Las funciones deben devolver sus resultados por consola para verificar los resultados.
